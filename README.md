# banco-de-dados

<p>Nesse projeto eu e meus colegas de sala, criamos um banco de dados totalmente voltado para estudos. O projeto consiste em uma empresa ficticia que funciona como se fosse uma agência de viagens, tendo como modelo de negócio B2B, vendendo apenas para empresas. As empresas solicitam viagens para nós e nós quem solicitamos para a empresa, tirando todo tipo de burocraia do nosso cliente(empresa), fazendo com que o cliente apenas viaje.</p>

<p>Nessa nossa empresa ficticia, tem como objetivo facilitar processos que demoram muito tempo pela quantidade de funcionários dentro da mesma.</p>

<p>O tipo de tecnologia utilizada nesse projeto foi MySQL usando o XAMPP para poder rodar o nosso banco de dados.</p>
